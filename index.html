Bernini Team

GitHub Tech Report

Overview

Bernini is a unified framework for video generation and editing that connects multimodal large language models with diffusion models. An MLLM-based planner predicts semantic plans in the ViT embedding space, while a DiT-based renderer turns these plans into high-quality videos using text and visual features.

With semantics as the interface, Bernini preserves the pretrained strengths of both components and enables efficient training. It introduces Segment-Aware 3D RoPE and planner chain-of-thought reasoning, achieving state-of-the-art results across video generation and editing benchmarks.
